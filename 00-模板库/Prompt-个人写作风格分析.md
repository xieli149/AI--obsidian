# Prompt-个人写作风格分析

## 使用位置

用于让 AI 深度学习历史文章，生成个人写作风格分析报告。

## 使用前确认

- 历史文章素材位置：`03素材库/01-历史文章素材`
- 建议素材数量：至少 5 篇，最好 10-20 篇
- 素材要求：尽量选择自己写过、阅读量较好、能代表表达方式的公众号文章
- 本 Prompt 已基于真实历史文章（4篇代表作）优化，风格结论可直接引用

## Prompt

```text
# 核心任务

请你深度学习我知识库中「03-素材库/01-历史文章素材」文件夹里的所有历史文章，
全面、精准地分析我的个人写作风格，形成一份详细的写作风格分析报告，
保存为独立笔记文件，命名为「我的个人写作风格分析报告.md」，保存路径为「00-模板库」。

# 分析维度（必须全部覆盖，不能遗漏）

1. 标题风格：分析我的标题常用格式、关键词、语气、吸引用户的方式。
   结合真实历史文章标题，总结 3 个我最常用的标题公式，每个公式给出具体示例。

2. 开头风格：分析我常用的开头方式（痛点引入、故事引入、金句引入、提问引入等）。
   总结开头逻辑、常用句式、用词特点，每个维度引用真实文章段落作为佐证。

3. 行文结构：分析我常用的文章结构（总分总、递进式、并列式、教程+感悟混合等），
   每个部分的篇幅占比、段落划分习惯、每段字数特点。

4. 用词习惯：分析我常用的词汇、句式、语气（口语化/书面化、轻松/专业），
   有没有常用口头禅、语气词、连接词。

5. 语气语调：分析我文章的整体人设和语气，
   是过来人前辈、平等的朋友，还是陪伴型助手？
   温柔治愈、犀利直接、还是科普提醒型？
   有没有固定的人设特点和口头表达方式。

6. 金句特点：分析我常用的金句格式、特点，
   喜欢在文章的哪个位置放金句，金句风格是怎样的，
   总结 2 个我常用的金句公式。

7. 结尾风格：分析我常用的结尾方式（总结升华、互动提问、行动指引、情感共鸣等），
   总结结尾逻辑和常用句式。

8. 排版习惯：分析我文章中使用的排版元素，
   包括 emoji 使用频率和位置、是否使用分隔线、标题层级、列表样式、加粗/引用等。

9. 其他特点：分析我文章中其他独有的、区别于其他人的写作特点，
   比如喜欢用案例、喜欢分点、喜欢用个人故事等。

# 输出要求

1. 分析必须精准、具体，基于我提供的历史文章，不能泛泛而谈；
   每个分析维度都要有对应案例，从历史文章中截取对应内容作为示例。
2. 最终形成一份完整的「个人写作风格分析报告」，
   严格按照以上 9 个维度，用 Markdown 格式输出，结构清晰，逻辑连贯。
3. 这份报告将作为后续为我生成文章的唯一风格标准，
   必须全面、精准、可落地。
4. 报告保存为独立文件「00-模板库/我的个人写作风格分析报告.md」。
```
