````Markdown
**查询1:待创作选题清单**

## 📋 待创作选题清单

自动列出所有待创作的选题，按优先级从高到低排序。

```dataview
TABLE WITHOUT ID
  file.link AS "选题标题",
  选题类型 AS "类型",
  创作优先级 AS "优先级",
  创建日期 AS "创建日期"
FROM "01-选题库"
WHERE 选题状态 = "待创作"
SORT 创作优先级 DESC, 创建日期 ASC


> **效果说明：** 这个查询会自动扫描「01-选题库」下所有笔记的YAML字段，筛选出「选题状态」为"待创作"的笔记，以表格形式展示标题（可点击跳转）、类型、优先级和创建日期。当你在选题库中新增或修改选题状态后，表格会自动更新。

**查询4：最近7天新增选题（辅助选题决策）**

```Markdown
## 🕐 最近7天新增选题

```dataview
TABLE WITHOUT ID
  file.link AS "选题标题",
  选题类型 AS "类型",
  创作优先级 AS "优先级",
  创建日期 AS "创建日期"
FROM "01-选题库"
WHERE 创建日期 >= date(today) - dur(7 days)
SORT 创建日期 DESC


> **效果说明：** 自动筛选最近7天内创建的选题，帮助你快速回顾近期的选题灵感，辅助决策本周优先创作哪些内容。
````
