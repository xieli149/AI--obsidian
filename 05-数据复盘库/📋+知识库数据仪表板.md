#+📋+知识库数据仪表板

##+📋+待创作选题清单

自动列出所有待创作的选题，按优先级从高到低排序。

```dataview
TABLE+WITHOUT+ID
++file.link+AS+"选题标题",
++选题类型+AS+"类型",
++创作优先级+AS+"优先级",
++创建日期+AS+"创建日期"
FROM+"01-选题库"
WHERE+选题状态+=+"待创作"
SORT+创作优先级+DESC,+创建日期+ASC
```

>+效果说明：这个查询会自动扫描「01-选题库」下所有笔记的+YAML+字段，筛选出「选题状态」为“待创作”的笔记，以表格形式展示标题、类型、优先级和创建日期。

##+✍️+稿件创作进度

自动列出所有创作中的稿件及其文件夹路径。

```dataview
TABLE+WITHOUT+ID
++file.link+AS+"稿件文件",
++file.folder+AS+"所在文件夹"
FROM+"02-稿件库/01-创作中稿件"
SORT+file.ctime+DESC
```

>+效果说明：自动扫描「02-稿件库/01-创作中稿件」下的所有文件，显示每个文件及其所在文件夹。

##+📊+知识库数据总览

```dataviewjs
//+选题库统计
const+allTopics+=+dv.pages('"01-选题库"');
const+pending+=+allTopics.where(t+=>+t.选题状态+===+"待创作").length;
const+creating+=+allTopics.where(t+=>+t.选题状态+===+"创作中").length;
const+completed+=+allTopics.where(t+=>+t.选题状态+===+"已完成").length;
const+total+=+allTopics.length;

//+稿件库统计
const+drafts+=+dv.pages('"02-稿件库/01-创作中稿件"').length;
const+published+=+dv.pages('"02-稿件库/02-已发布稿件归档"').length;

//+素材库统计
const+materials+=+dv.pages('"03-素材库"').length;

//+日常素材统计
const+dailyMaterials+=+dv.pages('"04-日常素材"').length;

//+渲染统计表格
dv.table(["统计项",+"数量"],+[
++["📚+选题总数",+total],
++["⏳+待创作选题",+pending],
++["✍️+创作中选题",+creating],
++["✅+已完成选题",+completed],
++["📝+创作中稿件",+drafts],
++["📤+已发布稿件",+published],
++["📦+素材总数",+materials],
++["📥+待分类日常素材",+dailyMaterials]
]);
```

>+效果说明：使用+DataviewJS+自动统计选题库、稿件库、素材库和日常素材的数据，生成实时更新的数据总览表。

##+🕐+最近7天新增选题

```dataview
TABLE+WITHOUT+ID
++file.link+AS+"选题标题",
++选题类型+AS+"类型",
++创作优先级+AS+"优先级",
++创建日期+AS+"创建日期"
FROM+"01-选题库"
WHERE+创建日期+>=+date(today)+-+dur(7+days)
SORT+创建日期+DESC
```

>+效果说明：自动筛选最近+7+天内创建的选题，帮助你快速回顾近期选题灵感。

##+内联查询示例

除了在代码块中创建完整查询，你还可以在任意笔记的正文中使用**内联查询**，实时显示某个字段值：

```text
当前笔记的创建日期是：`=+this.file.ctime`
当前笔记所在文件夹：`=+this.file.folder`
当前笔记的选题状态：`=+this.选题状态`
```

>+效果说明：内联查询会在笔记正文中直接显示查询结果，而不是生成表格。使用前要确保+Dataview+设置中的+Enable+Inline+Queries+已开启。
