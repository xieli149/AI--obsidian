# 📋 知识库数据仪表板

> 实时统计你的内容创作数据，一目了然

---

## 📊 查询1：知识库数据总览

```dataviewjs
// 选题库统计
const allTopics = dv.pages('"01选题库"');
const pending = allTopics.where(t => t.选题状态 === "待创作").length;
const creating = allTopics.where(t => t.选题状态 === "创作中").length;
const completed = allTopics.where(t => t.选题状态 === "已完成").length;
const total = allTopics.length;

// 稿件库统计
const drafts = dv.pages('"02稿件库/01-创作中稿件"').length;
const published = dv.pages('"02稿件库/02-已发布稿件归档"').length;

// 素材库统计
const materials = dv.pages('"03素材库"').length;

// 日常素材统计
const dailyMaterials = dv.pages('"04-日常素材"').length;

// 渲染统计表格
dv.table(["统计项", "数量"], [
  ["📚 选题总数", total],
  ["⏳ 待创作选题", pending],
  ["✍️ 创作中选题", creating],
  ["✅ 已完成选题", completed],
  ["📝 创作中稿件", drafts],
  ["📤 已发布稿件", published],
  ["📦 素材总数", materials],
  ["📥 待分类日常素材", dailyMaterials]
]);
```

> **效果说明：** 自动统计选题库、稿件库、素材库的各项数据，生成一个一目了然的统计表格。所有数据实时更新，无需手动维护。

## 📋 查询2：待创作选题清单

自动列出所有待创作的选题，按优先级从高到低排序。

```dataview
TABLE WITHOUT ID
  file.link AS "选题标题",
  选题类型 AS "类型",
  创作优先级 AS "优先级",
  创建日期 AS "创建日期"
FROM "01选题库"
WHERE 选题状态 = "待创作"
SORT 创作优先级 DESC, 创建日期 ASC
```

> **效果说明：** 自动扫描「01选题库」下所有笔记的 YAML 字段，筛选出「选题状态」为"待创作"的笔记，以表格形式展示标题（可点击跳转）、类型、优先级和创建日期。当你在选题库中新增或修改选题状态后，表格会自动更新。

## ✍️ 查询3：稿件创作进度

自动列出所有创作中的稿件及其文件夹路径。

```dataview
TABLE WITHOUT ID
  file.link AS "稿件文件",
  file.folder AS "所在文件夹"
FROM "02稿件库/01-创作中稿件"
SORT file.ctime DESC
```

> **效果说明：** 自动扫描「02稿件库/01-创作中稿件」下的所有文件，显示每个文件及其所在文件夹。你可以快速了解当前有哪些稿件正在创作中。当稿件标记为已发布并移动到「02-已发布稿件归档」后，此列表会自动移除对应条目。

## 🕐 查询4：最近7天新增选题

```dataview
TABLE WITHOUT ID
  file.link AS "选题标题",
  选题类型 AS "类型",
  创作优先级 AS "优先级",
  创建日期 AS "创建日期"
FROM "01选题库"
WHERE 创建日期 >= date(today) - dur(7 days)
SORT 创建日期 DESC
```

> **效果说明：** 自动筛选最近7天内创建的选题，帮助你快速回顾近期的选题灵感，辅助决策本周优先创作哪些内容。

## 💡 查询5：内联查询示例

在任意笔记中实时显示字段值：

| 用途 | 语法 |
|------|------|
| 创建日期 | `= this.file.ctime` |
| 所在文件夹 | `= this.file.folder` |
| 选题状态 | `= this.选题状态` |
| 创作优先级 | `= this.创作优先级` |
